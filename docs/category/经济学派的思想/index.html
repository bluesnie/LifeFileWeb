    经济学派的思想

    思想泡泡

    现代经济学的奠基者，是苏格兰哲学家亚当斯密。他于1776年出版了一本名为《国富论》的著作，这本书系统讨论了生产、市场和经济学理论，拥有无可比拟的影响力。斯密提出观点，一个人按照自己的个人利益行动最终会造福其他人。它同样提倡自由贸易，当时的许多国家都有沉重关税，牺牲贸易来保护他们的本土制造业。一代人以后，英国经济学家大卫李嘉图扩展了斯密的观点。提出了相对优势理论，理论认为，两个人或国家都可以从贸易中获益，哪怕其中一方可以生产更多的任何东西。当人们集中生产自己最擅长的东西，然后交换，每个人都能获益。总之，经济学领域继续发展，提出了例如私人财产和自由市场的观点。然后就出现了1848年的《共产党宣言》。不关注个体的行为，德国哲学家卡尔马克思和弗里德里希恩格斯研究经济阶级，认为历史是由工人和资本家的矛盾书写的，这个过程不可避免的引起工人推翻自己的老板，建立一个新的无国家无阶级的体系，共产主义。马克思由此写了《资本论》，马克思主义经济学产生的政治运动，挑战亚当斯密认为个人追求自身利益会造福社会的观点，最后的结果就有了两大阵营，自由市场资本主义支持私有财产和共产主义提倡生产资料的集体所有制。
