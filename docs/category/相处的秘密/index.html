    1．不要用网红恋爱观要求对方

    比如"分享欲是最高级的浪漫"，于是认为没给你天天发消息对方就不爱你了？

    成年人的世界里不是只有恋爱，还有事业家人朋友，健康，可能他只是这段时间忙些，或他不习惯发消息聊天，更喜欢面对面交流。
