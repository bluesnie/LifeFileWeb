    收入不平等

    经济不平等主要有两大类型，财富不平等和收入不平等。 财富是积累的资产减去负债，财富包括存款、养老金、不动产和股票这类东西的价值。 当我们谈到财富不平等时，我们基本上是在说资产分配的不平等。收入是不断被添加进财富中新的所得，所以当我们谈到收入不平等时，我们是在说新物质的分配不平等。

    思想泡泡

    让我们在全球水平看一看这两种类型的不平等。 现如今，全球财富估计大约有260万亿美元，但并没有均等分配。 一项研究表明，北美和欧洲，尽管拥有的人口不足全世界的20%，却拥有了70%的世界财富，中国人口比北美和欧洲加起来还多，却只有8%的世界财富。印度和非洲加起来几乎构成了世界人口的30%，但只拥有大约2%的世界财富。 我们现在重点放在收入不平等上，假设有10个人代表了地球上的每一个人，他们按照收入大小排成1排，最穷的在左边，最富的在右边，第一二位代表了最穷的 20%，第三四位代表了第二穷的20%，以此类推，如果我们把100美元，按照目前的收入趋势分配，最后两个人会得到83美元，第二富的这两个人获得10美元，中间的两位获得4美元，第二穷的这群人获得2美元，最穷的20%的人会得到1美元。 布兰科米兰诺维克一名专门研究不平等的经济学家，用经“济学大爆炸”来解释所有这些现象，一开始不同国家的收入都是差不多的，但随着工业革命的开始，差异就开始爆炸，这种爆炸推动一些国家迈入了高速的行列，而其他国家人在原地不动保持了几千年，根据米兰诺维克的想法，在1820年，世界上最富有的国家英国和荷兰的财富只是最穷的国家的3倍，如今最富国家和最穷国家的差距大概是100：1，这差距还仍然在越拉越大。
